# React for Dummy

### Quick Start

```sh
npm   install -g bower                                  # may require "sudo" (skip if you already have bower)
bower install                                           # installing "front end" dependencies
npm   install -g babel-cli                              # may require "sudo" (skip if you already have babel-cli)
npm   install                                           # installing jsx compiler
babel --presets es2015,react --watch src/ --out-dir js/ # compile jsx file
```
